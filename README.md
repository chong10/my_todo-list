# my_todo-list



설정을 누르면 배경화면 바꾸는 버튼이 있습니다.    
todo의 배경화면이 될 것입니다.    


마감시간은 사용자가 생각하는 오늘 할 일의 마감시간입니다.    
동작중인 todo는 시간을 멈추고, 미완료된 남은 일정과 함께 내일 일정으로 넘어갑니다.     
나머지 완료된 일정은 마감시간 종료 알림창과 함께 어떤 일정을 끝냈는지, 시간은 전체 비율 중 얼마나 썼는지 통계로 확인 할 수 있습니다.
